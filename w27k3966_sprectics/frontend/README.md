│   │    └─ 비밀번호 변경
│   │        └─ API: PUT /users/<user_id>

│        └─ API: DELETE /users/<user_id>

│        ├─ 목록 조회 (API: GET /posts)

│        │

│             ├─ 댓글 작성 (API: POST /comments)
